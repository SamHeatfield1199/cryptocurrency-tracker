# Cryptocurrency Tracker

Cryptocurrency tracker with the ability to register and track selected cryptocurrencies.<br/>
Data provided by [Coin Gecko API](https://www.coingecko.com/en/api)

### Build with:
- ReactJS
- ChartJS
- Material UI
- ContextAPI
